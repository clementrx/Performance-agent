# Example programs

What PerformanceAgent actually writes to `athlete/programs/program-v1.md` — one file
per goal from the README examples, in five languages. Feasibility probabilities and
week-by-week periodization factors were computed by the deterministic engine; every
citation comes from the verified evidence corpus. Athlete-specific details (equipment,
available days) are illustrative.

The determinants engine also coaches sports with **no seed model** — the coach
researches the event and fills a structured PerformanceModel, and the same tools do
the rest. Worked example: [K1 200 m sprint (unseeded)](en/kayak-sprint-unseeded.md).
